In the Main Menu:

	Level Editor - Goes to the level editor, the main function of this game
	Exit - Closes the program



In the Level Editor:

	S - Open the save menu
		Click on a slot to save to that slot
		L- Open the load menu
		Click on a slot to load from that slot

	Esc - Open the menu
		Save - Opens the save menu
		Load - Opens the load menu
		Clear - Resets the level to the "ground" tile
		Exit - Returns to the main menu

	Tab - Opens the tile menu
		Clicking on a tile will allow you to draw that tile on the map
